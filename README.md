# mktdata
A couple routes to retrieve put-call ratios and expense ratios for assets.

### Example usage
https://mktdata.fly.dev/er/VTI
https://mktdata.fly.dev/pcr/vol/VTI/30
https://mktdata.fly.dev/pcr/oi/VTI/10


### Example input
https://mktdata.fly.dev/pcr/oi/VTI/10
```
pcr
oi
VTI
10
```


### Example output
https://mktdata.fly.dev/pcr/oi/VTI/10
```
2.0982
```
